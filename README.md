### Development
1. Copy `.env.example` to `.env`
2. Fill out the required tokens in `.env`
3. Change the config files in `config/` as required
4. `virtualenv venv && source venv/bin/activate`
5. `pip3 install -r requirements.txt`
4. Good to go!
